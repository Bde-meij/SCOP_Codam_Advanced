# notes
opengl is een library met prototypes welke ingevuld dienen te worden
zo wordt de connectie gemaakt met hardware

werkelijke implementaties worden meestal gemaakt door g-kaartmakers
een driverupdate van de graphische kaart is dus eigenlijk dat deze
implementatie wordt geupdate. zon update kan er voor zorgen dat een game
wel of niet werkt.

Khronos host specificatiedocumenten van openglversies (met zonder implementatiecode)

openGL is een statemachine (logische paden, 01001011, vooraf vastgelegd)
je kunt deze state manipuleren. de huidige state wordt de openGL context genoemd
je zou functies kunnen indelen in state-changing en state-using
